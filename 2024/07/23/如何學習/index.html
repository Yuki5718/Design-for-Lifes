如何學習

關於這篇文章的誕生

昨天專題的依依助教看完上週的進度回報後，希望我能分享如何學習。
恰巧今天中午吃飯時，在FB上的新聞媒體分享了「奕維先生」這位youtuber的影片。
看影片的同時，我認同這些觀點，也和我平常的學習方法非常相似
也整理了其他的我會使用的學習方法

PART 1

我為影片大致上做了整理

最沒效率的學習方式

  1. 不重複閱讀　→　不等於不複習　→　底層邏輯
  2. 畫重點　　　→　不是邊讀邊畫　→　學會之後，提煉精華
  3. 作筆記　　　→　不是邊讀邊寫　→　學會之後，提煉精華

怎麼讀?

  1. “知道”資訊並不是”記憶”。
    接收新知識時，就要分門別類，要帶著好奇、從中獲得樂趣
  2. 學習層次目標：記憶→理解→應用→分析→評估→創造

如何複習?

  1. 塗鴉法　→　捋清思緒
    (看目錄→去思考，寫在塗鴉紙上)
  2. 教學法　→　教學相長
    (教一個不懂的人)　(費曼學習法)
  3. 寫考古題　→　做中學

如何休息?

  1. 感覺想休息時，就休息。(發現會分心時)
  2. 休息時不能接收新資訊
  3. 睡覺>放空+靜坐+冥想>起來活動+做家事

手機影響巨大!!

手機收在不同地方 手機放桌上會嚴重影響學習效率

如何規劃?

先讀重點，提早讀完(在規劃的時間內一半讀完)
規劃大框架就好，別規劃太細

PART 2 認知負荷

核心重點：把複雜的大東西，切分成小碎塊，再利用小碎塊重組

工作記憶：短時記憶，暫存
長時記憶：模塊化 → 前知識
理解＝工作記憶、長時記憶的協作
刻意練習 → 累積前知識 → 加速工作記憶調動長時記憶的能力

祛除冗餘 → 祛除多餘的干擾，Ex：IEKA組裝說明書，純圖片
分割複雜 → 複雜切分為小塊，再重組
交替實例 → 練習相似題型

PART 3 費曼學習法 (教學相長，學以致用)

費曼學習法 - 四大步驟

  1. 確認學習目標
  2. 理解並教授它給他人
  3. 回顧並反思
  4. 簡化和吸收

我的作法

覺得學會了就嘗試教人
過程中就是在腦內整理，思考如何把知識濃縮精煉，加強自身對知識的印象
如果做不到，就是學的還不夠透徹
被教學的對象提出的問題擊敗了，就代表還有要繼續學習的弱項
不斷反覆這個過程，自己就會對這個知識越熟悉、越透徹
